
# 📒 Contact Hub Pro (Tkinter GUI App)

A simple and user-friendly Contact Management System built using Python Tkinter.  
This application allows users to store, search, update, and manage contacts with a clean GUI interface.

---

## 🚀 Features

- ➕ Add new contacts
- 📋 View all contacts in table format
- 🔍 Search contacts by name, phone, or email
- 🗑 Delete contacts
- ⭐ Mark contacts as favorites
- 💾 Data saved automatically using JSON file
- 📊 Dashboard showing total and favorite contacts

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- JSON (Data Storage)
